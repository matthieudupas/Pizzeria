Ce projet est une simulation complète de la gestion d’une pizzeria, développée dans un objectif pédagogique : illustrer l'utilisation concrète de plusieurs patrons de conception (design patterns) dans un même projet structuré.

L’application couvre l’ensemble du processus métier d’une pizzeria : prise de commande, préparation des pizzas, facturation et suivi des stocks — le tout en mettant en œuvre des patterns orientés objet pour une architecture propre, extensible et maintenable.
